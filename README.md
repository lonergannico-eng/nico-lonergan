# Nicholas “Nico” Lonergan — GitHub Pages Website

## Files
- `index.html` — main website page
- `styles.css` — styles
- `script.js` — mobile menu and footer year
- `assets/` — photos, CV, and poster PDF

## To publish on GitHub Pages
1. Open your GitHub repository.
2. Upload **all files and folders inside this website folder**.
3. Make sure `index.html` stays in the top level of the repository.
4. In GitHub: **Settings → Pages**.
5. Under **Build and deployment**, set the source to **Deploy from a branch**.
6. Select **main** branch and **/(root)** folder.
7. Save.
8. Wait about 1 minute, then refresh your site URL.

## Easy edits
- Update text in `index.html`
- Replace photos in `assets/`
- Replace the poster PDF in `assets/AGU_Poster_Lonergan.pdf`
- Update links in the About / Contact sections
